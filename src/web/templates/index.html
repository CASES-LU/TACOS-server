TACOS is the mobile application by CASES.LU and stands for TrustApp Cases for Obvious Security.

TACOS provides a spam detection feature. It allows you to check for spam phone numbers. You can also contribute to our database by reporting spam phone numbers.

Besides that, TACOS is also an awareness application for cybersecurity. We included a password cards generator, some security and privacy awareness games.

Features
